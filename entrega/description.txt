La aplicación web “Stacks” es una red social diseñada específicamente para jugadores de videojuegos que buscan formar equipos o encontrar compañeros de juego con niveles de habilidad similares. Es sumamente complicado encontrar jugadores adecuados para formar equipo en títulos competitivos como Valorant u Overwatch. Mi aplicación propone una solución accesible y especializada dentro del creciente entorno de los videojuegos y los e-sports, facilitando la conexión entre jugadores con intereses y objetivos compatibles.

El funcionamiento general de Stacks gira en torno a un sistema de perfiles de usuario donde cada jugador puede registrarse, personalizar su información y realizar búsquedas avanzadas mediante filtros como el género y edad de los integrantes dentro del equipo, el rango o el modo de juego preferido. La plataforma incorpora herramientas sociales como mensajería interna entre equipos, anuncios públicos para buscar compañeros y una sección de noticias relacionadas con el ámbito gamer. Estas funciones están enfocadas en mejorar la experiencia de usuario, fomentar la interacción y agilizar el proceso de creación de equipos.

Desde el punto de vista técnico, Stacks se desarrolla utilizando tecnologías modernas orientadas a la escalabilidad y eficiencia. El frontend está construido con React y estilizado con Tailwind CSS, proporcionando una interfaz ágil y visualmente atractiva. El backend está basado en Node.js, y la persistencia de datos se gestiona mediante MongoDB a través de Mongo Express. Todo el sistema se despliega en contenedores Docker para garantizar portabilidad y facilidad de mantenimiento, con Nginx como proxy inverso para gestionar el tráfico. Además, se han añadido tecnologías complementarias como los feeds RSS, que permiten a los usuarios obtener noticias o anuncios actualizados diariamente. También, se han implementado certificados SSL proporcionados por Let's Encrypt.
